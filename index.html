    NYC BIKE APP

    This app is a must-have for anyone that regularly uses the NYC Bike, particularly if they use an array of routes. Not only will it help you plot a docking station in any given area, it will also tell you how many bikes and spaces there are on an easy-to-follow map. You can also use the app to view your NYC Bike account details, including recent journeys.

  • This app allows the user to find and manage bike rides. It provides real time information about available bikes and docking stations in NYC leveraging information from public API.
  • It has features to select appropriate stations as well as pictorial view leveraging Google location API.
  • It also helps the user to manage his rides by posting his rides and allowing with features to post, modify or delete his rides.
  • It also provides information to the user about rules of the road and how to ride bike and safety rules.
  • It also gives the user information about various places of interest for the bike ride as well as for tourism purposes.
